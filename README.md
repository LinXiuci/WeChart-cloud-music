# 仿网易云音乐（小程序版）


### 通过对网易云音乐API解析整合而成的在线音乐播放平台。提供歌曲、mv 播放、电台播放、搜索、榜单等功能。

### 模块
1. 发现（swpier、每日推荐、歌单、榜单）
2. 视频（mv 播放）
3. 我的（登录、最近播放、喜欢、创建的歌单、收藏的歌单）
4. 播放器（播放列表、上下一首、歌词跟踪、收藏、喜欢）
